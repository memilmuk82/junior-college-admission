# ADR-0001: PostgreSQL 기본 데이터베이스

- 상태: 채택
- 결정일: 2026-07-10

## 결정

개발과 운영의 기본 데이터베이스를 PostgreSQL로 통일한다. `web`과 PostgreSQL `db`는 별도 컨테이너로 실행하고, DB 영속 데이터는 `postgres_data` named volume에만 저장한다.

## 근거

- 개발·운영 간 데이터베이스 동작 차이를 줄인다.
- DB 파일을 애플리케이션 이미지와 작업 트리에서 분리한다.
- 자격증명은 런타임 `.env`에서 주입하고 이미지에 포함하지 않는다.

## 영향

- 기준 문서의 개발용 SQLite 방향은 2026-07-10 사용자 지침으로 대체한다.
- 단위 테스트의 격리 전략과 마이그레이션은 Phase 1에서 확정한다.
