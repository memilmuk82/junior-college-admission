# Phase 7 대학 규칙 검수 후보

이 문서는 사용자 확인 내용과 읽기 전용 참고자료를 관리자 검수 후보로 정리한다. 실제 seed, `HUMAN_APPROVED`, `PUBLISHED` 규칙이 아니며 공식 PDF 페이지 대조가 완료되지 않은 필드는 추정하지 않는다.

## 공통 변환 계약

- Z점수 등급 경계 `1.76`, `1.23`, `0.74`, `0.26`, `-0.26`, `-0.74`, `-1.23`, `-1.76`과 각 포함 방향은 기존 `Z_SCORE` 표 계약 및 경계 테스트로 표현 가능하다.
- `원점수·평균·표준편차 → Z점수 → 석차등급`은 `STANDARD_Z_V1`로 표현하되 대학 공식 PDF 표의 반올림·절단·경계를 우선한다.
- A/B/C를 1/3/5등급으로 바꾸는 규칙은 버전 고정 `GRADE_TABLE` 후보이며 자유 수식으로 저장하지 않는다.
- 성취도 누적분포 등급은 `CUMULATIVE_DISTRIBUTION_GRADE_V1` 후보이며 A/B/C 고정 1/3/5와 혼동하지 않는다.

## 관리자 검수 시 필수 확인

- 정확한 모집학년도·캠퍼스·수시 차수·전형 코드
- 직업위탁 학생의 지원자격과 원적교/위탁기관 성적 범위
- 3학년 1학기 포함 여부
- 학기별 이수단위 가중과 우수학기 동점 처리
- Z점수 반올림 시점, 절단 범위, 경계 포함 방향
- 진로선택이 Z점수·누적분포·고정 1/3/5 중 어느 계약인지
- 교과 환산점수, 출결, 면접·실기의 분리
- 최종 모집요강 정정본 존재 여부

AI는 이 후보의 사람 승인이나 게시 상태를 설정하지 않는다.
