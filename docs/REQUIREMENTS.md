# 요구사항 기준

요구사항 원문은 저장소 루트의 다음 문서를 그대로 사용한다.

1. `2027_전문대_입시상담앱_실행개발문서_v2.md`
2. `CODEX_MASTER_PROMPT_2027_전문대_입시상담앱_v3.md`

v3는 v2를 상위 실행 명세로 명시한다. Phase 0에서는 저장소·자료 안전화를 우선하며, 최소 실행 화면은 도메인 기능이 없는 상태 확인용 앱 셸로 제한한다.


## 사용자 확정 지침

- 2026-07-10: 개발·운영 기본 DB는 PostgreSQL을 사용한다.
- `web` 컨테이너, `db` 컨테이너, DB 저장 volume을 분리한다.

## 프론트엔드 확정 지침

- Flask Jinja2 SSR을 기본으로 하며 SPA 프레임워크와 TypeScript를 사용하지 않는다.
- 핵심 기능은 JavaScript 없이 폼 제출과 서버 렌더링으로 동작한다.
- Vanilla JavaScript와 fetch API는 필요한 부분 상호작용에만 사용한다.
- 학생용·교사용 화면은 별도 A4 인쇄 CSS를 사용한다.
