# 공개 정제 전형 데이터

이 디렉터리에는 외부 공개가 가능하고 출처·학년도·규칙 버전·검수 상태를 확인한 정제 전형 데이터만 둘 수 있습니다.

원본 PDF·엑셀, 학생 개인정보, 사용자 업로드, OCR 파생물, 미검증 데이터는 둘 수 없습니다. 각 데이터 파일은 출처 식별자와 사람 검수 상태를 함께 가져야 합니다.

Phase 17 공개 seed는 원본 CSV를 포함하지 않습니다. `phase17_public_admission_catalog.csv`에는 공개 선택 기준정보 4,970행을 보존하고, `phase17_public_admission_results_2026.csv`에는 점수 척도와 범위가 검증된 4,094행만 두며, 제외 근거는 `phase17_public_admission_audit.json`에 기록합니다.
